Pour faire un site responsive :
Soit mettre dans le HTML un lien pour charger le CSS

          <link media="screen and (max-width: 480px)" href="css/smartphone.css" type="text/css" rel="stylesheet" />

Soit directement dans un CSS existant :

          @media screen and (max-width: 480px) {
                  body {
                              background : #FF0000;
                                  }
          }

Et on peut combiner : 
          @media screen and (min-width: 600px) and (max-width: 800px) and (orientation: portait) {

          }
